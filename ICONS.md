# Icon Files Required

This extension requires the following icon files in PNG format:
- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

## Options

1. **Create your own icons** - Design icons that represent recording/Google Meet
2. **Use placeholder icons** - Generate simple colored squares temporarily
3. **Remove icon references** - Edit `manifest.json` to remove icon references if you don't need them

## Quick Placeholder Icons

You can use any image editor to create simple placeholder icons:
- 16x16, 48x48, and 128x128 pixel PNG files
- Any color scheme that fits your branding
- Simple design with "R" or recording symbol

## Note

The extension will work without icons, but Chrome may show a default icon instead.
